## INSTALLATION

To install this module type the following:

   perl Makefile.PL
   make
   make test
   make install

## DEPENDENCIES

This module requires these other modules and libraries:

  Compress::Zlib
  Date::Format
  Digest::MD5
  POSIX
  Storable
  URI::Escape::XS
